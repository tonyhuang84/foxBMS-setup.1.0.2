Configuration of the GPIOs

Supported packages of the STM32F429

In io_cfg.h the users defines the used package of the STM32F429 by a macro. The following table shows the package and its corresponding macro:

Package Macro
LQFP100 IO_PACKAGE_LQFP100
LQFP144 IO_PACKAGE_LQFP144
LQFP176 IO_PACKAGE_LQFP176

The global enable for locking of pin configuration is set (Macro IO_PIN_LOCKING). Enabling this macro allows to specify in a locking variable in io_cfg[] array in io_cfg.c.
