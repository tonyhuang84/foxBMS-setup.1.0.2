◆ MEM_BKP_SRAM

#define MEM_BKP_SRAM   __attribute__((section (".BKP_RAMSection")))

A variable defined as (type) MEM_BKP_SRAM (name) will be stored in the RAM which is backuped by a button cell. Therefore as long as the power supply is not disconnected, the variable value will be stored during e.g. restarts.
