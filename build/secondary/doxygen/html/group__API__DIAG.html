DIAG_Handler provides generic error handling, based on diagnosis group.This function calls the handler functions depending on the diagnosis group of call. It needs to get called in every function which wants to apply some kind of diagnosis handling. According to its return value further treatment is either left to the calling module itself, or can be done in the callback function defined in diag_cfg.c.

Parameters
diag_ch_idevent ID of the event that has occurred
eventevent that occurred (OK, NOK, RESET)
item_nritem nr of event, to distinguish between different calling locations of the event
datapointer to data if DIAG_CONT_TYPE
Returns
DIAG_HANDLER_RETURN_UNKNOWN if invalid DIAG_TYPE_e, otherwise return value of DIAG_GeneralHandler or DIAG_ContHandler
