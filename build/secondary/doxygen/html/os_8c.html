◆ OS_TSK_Cyclic_100ms()

void OS_TSK_Cyclic_100ms ( void  )

cyclic 100ms-Task, preemptive with TSK_Cyclic_1ms() and TSK_Cyclic_10ms().

Task is delayed by a phase as defined in eng_tskdef_cyclic_100ms.Phase (in milliseconds). After the phase delay, the cyclic execution starts, the entry time is saved in currentTime. After one cycle, the Task is set to sleep until entry time + eng_tskdef_cyclic_100ms.CycleTime (in milliseconds). The task calls Job_100ms(), Job_500ms() and Job_1s().

◆ OS_TSK_Cyclic_1ms()

void OS_TSK_Cyclic_1ms ( void  )

cyclic 1ms-Task, preemptive with TSK_Cyclic_10ms() and TSK_Cyclic_100ms().

The Task calls OsStartUp() in the very beginning, this is the first active Task. Then the Task is delayed by a phase as defined in eng_tskdef_cyclic_1ms.Phase (in milliseconds). After the phase delay, the cyclic execution starts, the entry time is saved in currentTime. After one cycle, the Task is set to sleep until entry time + eng_tskdef_cyclic_1ms.CycleTime (in milliseconds). The Task calls Job_1ms(), Job_2ms() and Job_5ms().

Returns
